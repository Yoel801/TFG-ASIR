# Mi TFG de ASIR: Gestión de Incidencias con RFID

Esta es la web que desarrollé para mi proyecto final. Es un sistema para gestionar tickets de soporte que se activa con RFID y permite conexión remota.

**¿Cómo funciona?**
Aquí está el flujo principal del proyecto:

*   Un **Lector RFID** da una señal para generar la incidencia automáticamente.
*   Esa incidencia se guarda en la **Base de Datos** y sale en la página web.
*   A través de la web, te puedes conectar con un servidor **RustDesk** local usando el ID proporcionado.

**Otras funciones:**
También se puede usar la web para:

*   Generar nuevas incidencias manualmente.
*   Crear y gestionar nuevos usuarios.
*   Ver el registro histórico de incidencias.
