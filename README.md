**Código fuente de página web presentada en TFG de ASIR**

*En este código se muestra como crear incidencias de la siguiente forma:*

#- Un Lector RFID da una señal para generar la incidencia#
#- Esa incidencia es vista por la Base de Datos y la muestra en la página web#
#- Mediante un servidor Rustdesk local, puedes conectarte con el ID proporcionado*

*Además, puedes realizar las siguientes funciones:*
#- Generar nuevas incidencias#
#- Generar nuevos usuarios#
#- Revisar un histórico#
